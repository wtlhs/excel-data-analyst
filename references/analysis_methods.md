# 数据分析方法详解

## 1. 数据探索方法

### 1.1 数据概览
- **数据维度**: 行数、列数、文件大小
- **数据类型**: 数值型、文本型、日期型
- **缺失值分析**: 缺失比例、缺失模式
- **唯一值分析**: 基数、重复率

### 1.2 描述性统计
- **集中趋势**: 均值、中位数、众数
- **离散程度**: 标准差、方差、极差
- **分布形态**: 偏度、峰度
- **分位数**: Q1, Q2, Q3, IQR

## 2. 关联分析方法

### 2.1 列名相似度匹配
```python
from difflib import SequenceMatcher

def column_similarity(col1, col2):
    return SequenceMatcher(None, col1.lower(), col2.lower()).ratio()
```

### 2.2 值重叠分析
- 计算两列值的交集
- 计算重叠比例
- 识别潜在外键关系

### 2.3 相关性分析
- Pearson 相关系数（线性关系）
- Spearman 相关系数（单调关系）
- 互信息（非线性关系）

## 3. 异常检测方法

### 3.1 统计方法

**Z-Score 方法**
- 计算每个值与均值的偏差
- 标准差倍数 > 3 视为异常
- 适用于正态分布数据

**IQR 方法**
- Q1 - 1.5*IQR ~ Q3 + 1.5*IQR 为正常范围
- 不受极端值影响
- 适用于偏态分布

### 3.2 时间序列异常
- 移动平均检测
- 指数平滑检测
- 季节性分解

### 3.3 业务规则检测
- 预定义阈值告警
- 逻辑一致性检查
- 跨表一致性验证

## 6. 报告生成最佳实践

### 6.1 报告结构
1. **执行摘要** - 3-5 条核心发现
2. **数据概览** - 数据规模和质量
3. **关联分析** - 表格关系图
4. **核心指标** - 关键指标趋势
5. **风险预警** - 分级预警列表
6. **深度洞察** - 模式和规律
7. **行动建议** - 可执行建议

### 6.2 可视化建议
- 每个图表只传达一个核心信息
- 使用一致的配色方案
- 添加清晰的标题和标签
- 标注异常点和关键转折

### 6.3 文字说明
- 用数据说话，避免主观判断
- 提供具体数字和比例
- 解释异常原因
- 给出可执行建议
